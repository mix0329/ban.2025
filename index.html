    <div class="section" id="business-plan-overview">

      <h2>事業計画概要</h2>

      <h3>1. 目的（なぜそれをするのか）</h3>

      <ul>

        <li><strong>企業の存続と持続的成長の実現</strong><br>

          市場環境の厳しさを踏まえ、現状の「負のスパイラル」を断ち切り、全社が連動する正のスパイラル（好循環）を実現する。</li>

        <li><strong>組織の健全化と競争力強化</strong><br>

          正当な評価制度と透明な報酬体系を基盤とし、従業員のモチベーションを高め、サービス品質と店舗運営力の向上を図る。</li>

      </ul>

      <h3>2. 目標（何を達成するのか）</h3>

      <ul>

        <li><strong>営業利益予算 1億300万円 の達成</strong></li>

        <li><strong>根拠</strong>：BANは「強い店舗の集合体」を目指す。強いお店の定義は、顧客満足度が高い、給与が一般の1.5倍、そして営業利益が3倍という基準に基づく。</li>

        <li><strong>従業員と顧客の満足度向上</strong>

          <ul>

            <li>従業員：AIを活用した評価制度により店長・スタッフの成長促進と離職率低下</li>

            <li>顧客：定期アンケートによるサービス改善</li>

          </ul>

        </li>

      </ul>

      <h3>3. 手段（どのように実施するのか）</h3>

      <ul>

        <li><strong>評価・報酬制度の改革</strong>

          <ul>

            <li>AI活用の評価制度：自動査定とリアルタイムフィードバック</li>

            <li>正当な報酬体系：店長給50万円、アルバイト時給最大2000円／平均1700円</li>

          </ul>

        </li>

        <li><strong>現場力強化のための視察活動</strong>

          <ul>

            <li>毎月の社員・店長による繁盛店視察</li>

            <li>毎月のアルバイト視察</li>

            <li>年4回の地方視察（長崎、福岡、沖縄、高知）</li>

          </ul>

        </li>

      </ul>
